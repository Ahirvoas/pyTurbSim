TurbGen.phaseModels.base module¶

This is the base module for phase models. Phase models produce the initial phase information that is then correlated in the stress and coherence models.

The basic phase model simply returns a ‘random’ array.

class TurbGen.phaseModels.main.Uniform[source]¶

Bases: TurbGen.phaseModels.base.phaseModelBase

The uniform random phase model

This phase-model randomizes the phases uniformly between 0 and 2*pi. It has no ‘phase/temporal coherence’ (correlation as a function of frequency).

Because it is so simple, it has no initialization parameters.
