Calculate the *comp* (0, 1 or 2) coherence between points *ii*=(iz,iy) and *jj*=(jz,jy) for the IEC model.

This method is only used if tslib is not available.

Parameters ---------- *cohi* - A 'coherence calculator' instance (for the given tsrun). *comp* - an integer (0,1,2) indicating the velocity component for which to compute the coherence. *ii*,*jj* - Two-integer elements indicating the grid-points between which to calculate the coherence. for example: ii=(1,3),jj=(2,3)
