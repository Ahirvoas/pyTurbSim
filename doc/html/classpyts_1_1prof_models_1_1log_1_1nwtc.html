Detailed Description

The NWTC logarithmic mean wind-speed profile.

ln( z / Z0 ) - psi_M Ubar(z) = URef * ------------------------ ln( ZRef / Z0) - psi_M

Where psi_M is a function of the Ri, the Richardson number (psi_M=0 for Ri=0), and the turbulence model.
