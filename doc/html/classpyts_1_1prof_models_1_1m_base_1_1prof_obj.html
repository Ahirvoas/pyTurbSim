Detailed Description

Profile objects contain the array (self.array) of mean-velocity values for a specific PyTurbSim run.

This class defines various shortcuts to the mean velocity data including:

u - The u-component mean-velocity array v - The v-component mean-velocity array w - The w-component mean-velocity array

dudz - The vertical shear of u-component. dvdz - The vertical shear of v-component. dwdz - The vertical shear of w-component.

uhub - The hub-height u-component mean-velocity.
