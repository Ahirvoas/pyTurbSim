Detailed Description

The iec wind profile.

This profile is a power-law across the rotor disk and logarithmic elsewhere.

Parameters ---------- grid - The TurbSim grid object for this simulation. URef - Reference velocity for the wind profile [m/s]. RefHt - Reference height of the reference velocity [m]. PLexp - The power-law exponent to be utilized for this simulation [*non-dimensional], default=0.2 (per IEC specification). Z0 - Surface roughness length [m]. Ri - The Richardon number [*non-dimensional]. turbmodel- the name of the turbulence model in this simulationm, *optional*.

Returns ------- out - An 'IEC' wind profile object that matches the specified input parameters.
